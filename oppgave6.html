Git og GitHub

Git er en måte å dele eksempelkoder og det gjør det enklere at flere samarbeider om den samme koden. Grit er et verktøy for kodedeling og versjonskontroll. Det vil si at du kan lagre flere versjoner også gå tilbake i kodene, du får en oversikt over historikken og du har alltid backup.

GitHub er tjenesten som gir tilgang til Git versjonskontroll. Den lar deg se igjennom kodene som du eller andre har lagt inn.
